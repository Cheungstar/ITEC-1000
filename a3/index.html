2.  In general, there is a diminishing rate of return of expected increase in computing power when the number of cores (processors) grows large in a computer.   Explain coherently why one would expect this to be the case.  [10 marks]

Answer:

The most obvious way to increase the computing power and performance of a computer would be to increase the number of CPUs. Under ideal conditions, each CPU would possess its own assigned sequence of program instructions independently which would make a dual-core processor to process through double the instructions over a single-core processor in a given amount of time. In reality however, increasing the number of CPUs would be greatly effective up to a point, and after that point there would begin to have a diminishing rate of return in computing power when the number of cores grows even larger. When more and more CPUs get added into a computer, there would be a larger overhead that will be caused by the need to send different instructions to each CPUs as well as the CPUs need to share limited resources of the computer such as primary memory space, shared bus connections, and I/O devices. All these factors create the effect of the diminishing rate of return once the number of cores grows.

3.  Describe the advantages and disadvantages of mater-slave multiprocessing and symmetrical multiprocessing. Explain coherently why you would choose one over the other for fault-tolerant computing.  [10 marks]

Answer:

The master slave processing is a form of controlling a multiprocessing system through the assignment of a single CPU as the master of the entire system. This single CPU then governs and manages the entire system as well as controls and assigns resources and schedules. Any other CPU in the system is then called a slave which uses the resources its given by the master and completes the scheduled tasks. The advantage of the master-slave multiprocessing is that if a slave goes down there wont be much of an affect to the system and the system can keep on running. The master processor is also able to divide work up between the slave processors, and as each slave completes the tasks assigned and reports the results back to the master, the master is then able to piece together the final result which makes for an easier workload for each CPU. The disadvantage is if the master CPU is not a good strong CPU capable of performing many tasks and coordinating with slaves. Another disadvantage is that if the master CPU goes down, then the entire system will not function.

Symmetrical multiprocessing is when each and every CPU has identical access to the operating system and the system resources. Each CPU is able to schedule its own work and allocate its own resources to use within the parameters set by the operating system.  The advantage of this type of multiprocessing is to allow any processor to work on any task no matter where the data for that task is in memory and to be able to move tasks around to balance the workload efficiently.

In the case of fault tolerant computing, Symmetrical multiprocessing would be the better choice of the two. Any critical tasks can be distributed among the processors at once to complete faster. If one core shuts down, the entire system will not crash.

5.  Explain coherently why the average seek time for a hard disk much shorter than for a CD-ROM or DVD-ROM?  [5 marks]

Answer;

The average seek time for a hard disk is much shorter than a CD-ROM or DVD-ROMs average seek time because of the rotational speed a hard disk can spin at compared to a CD or DVD-ROM. With a hard disk, it has a rotations per minute (rpm) of 7200, while CD-ROMs and DVD-ROMs have a rpm of only 570 to 1600. Since it is many times less than a hard disk, the time it takes to get to the right block to access the information needed would take much longer than a hard disk. Also, a CD and DVD-ROM stores data on a single track instead of multiple tracks. This track is approximately three miles long and spirals from the inside of the disk to the outside with linear blocks acting as the sectors that would normally be found on a disk. The laser that finds the data in the CD or DVD-ROM will need more time than the head motor in a hard disk, resulting in CD and DVD-ROMs to have a longer average seek time than hard disks.

6.  Distinguish between pixel (raster) graphics and object (vector) graphics, and identify coherently the advantages and disadvantages of each.  [10 marks]

Answer:

Raster scan is the method to displaying all the pixels on a screen or monitor by scanning and displaying each pixel one row at a time from left to right and top to bottom. After one row has been scanned and displayed, the scanner then moves back to the left edge of the row below the row that was displayed and scans that row and so on. The advantage to raster scan is that it is used almost everywhere and cheaper than vector scan.

Vector scan is an alternative to the raster scan. This method displays pixels in whatever order is necessary in order to trace out the images. Vector scan is not suitable for bit map graphics but instead is used for object graphics image such as CAD/CAM applications. A disadvantage of vector scan is that it is expensive and more difficult compared to raster scanning.
